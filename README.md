# Criação de Ferramentas Básicas:

Repositório com aplicações desenvolvidas em **Python** utilizando **PySide6**, voltadas ao aprendizado de programação e desenvolvimento de interfaces gráficas.

## Aplicações:

### 🧮 Calculadora:
- Operações matemáticas básicas;
- Interface gráfica intuitiva;
- Ícone personalizado.

### 🔐 Gerador de Senhas:
- Geração de senhas aleatórias;
- Interface desenvolvida com **Qt Designer**;
- Definição da quantidade de caracteres;
- Ícone personalizado.

## Tecnologias usadas:

- Python
- PySide6
- Qt Designer
